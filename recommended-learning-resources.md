
### R Absolute Beginner

Here are the tutorials I recommend to folks learning R for the first time:


* Carpentries " Data Analysis and Visualization in R for Ecologists " **episodes 1&2 only**
  * https://datacarpentry.github.io/R-ecology-lesson/index.html
* Then switch to data.table tutorials to learn that system for data processing
  * Intro: https://r-datatable.com/articles/datatable-intro.html
  * Joins: https://r-datatable.com/articles/datatable-joins.html
  * Reshaping: https://r-datatable.com/articles/datatable-reshape.html

### Git
